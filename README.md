# Exercise_Unit5_website

Este repositorio contiene los archivos iniciales de un sitio web.

Para la página web utilizaremos dos archivos: un HTML para el código y un CSS para los estilos.

Para abrir el sitio web (index.html) en un navegador, debemos de crear el archivo y tenerlo localizado. Pulsamos clic derecho en 'index.html', abrir con y seleccionamos el navegador con el que queramos abrirlo.
